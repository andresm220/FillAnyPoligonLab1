# FillAnyPoligonLab1

This Rust project implements an algorithm to fill any polygon, as part of a computer graphics lab assignment.

##  Requirements

- [Rust](https://www.rust-lang.org/) (installed via [rustup](https://rustup.rs/))
- Windows with [WSL (Windows Subsystem for Linux)](https://learn.microsoft.com/en-us/windows/wsl/)
- An image viewer (e.g. Windows Photos)

##  Running the project

To compile and run the program:

```bash
cargo run

To view the image from WSL using Windows Explorer:

```bash

explorer.exe out.png
